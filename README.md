# ProjectLibrary Система за управление на библиотека
1. Цел на проекта на проекта е да се създаде система за управление на библиотека.
Проектът демонстрира използването на основните обектно-ориентирани принципи:
-наследяване
-полиморфизъм
-инкапсулация
-обработка на изключения

2.Структура на класовете
В проекта са използвани следните класове:
-User (абстрактен клас)
Има абстрактни методи:
-BorrowBook()
-ReturnBook()
Student
-Наследява класа User
-Може да заема и връща книги
Librarian
-Наследява класа User
-Може да добавя книги в библиотеката
-Не може да заема и връща книги
Library
-Съдържа списък с налични и заети книги
-Реализира логиката за:
-добавяне на книги
-заемане на книги

4. Реализирани функционалности
-Добавяне на книги от библиотекар
-Показване на налични книги
-Заемане на книга от ученик
-Връщане на книга от ученик
-Създаване на профили за потребители
-Обработка на грешки чрез изключения
